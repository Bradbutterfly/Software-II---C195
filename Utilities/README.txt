Title: Advanced Scheduling System
Purpose: The purpose of the application is to allow a user to schedule an appointment for customers.
This allows the user to add, modify, and delete the customer profiles. It also allows the user to add, modify, and delete appointments for the given customers.

IntelliJ: IntelliJ Community 2021.3.2
JDK: openJDK-17
JavaFX: JavaFX-SDK-17.0.1
MySQL driver: mysql-connector-java-8.1.28
Project language Level: SDK Default

Start application, Provide login credentials and execute. Once inside the application, use the given menu navigate to Customer, Appointment, Report, or Return.
Customer Screen goes to the customer menu to add, modify or delete, Add menu allows you to add customer with input data, Modify menu allows you to modify customer with input data, Delete menu allows you to delete customer and input data.
Appointment Screen goes to appointment menu to add, modify, or delete, Add menu allows you to add appointment with input data, Modify menu allows you to modify appointment with input data, Delete menu allows you to delete appointment and input data.
Reports screen goes to the area to view the reports available, Brief descriptions of reports are posted on the related tabs, Current reports offer, type and month, contact schedule, and current day appointment.
